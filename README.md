# Roadmap Engenharia de Automação e Infraestrutura 2026 🚀

Este repositório é um **PWA (Progressive Web App)** que serve como um hub interativo e offline para o dossiê técnico de primazia em automação, focado na desconstrução de frameworks e otimização de infraestrutura para ambientes complexos.

## ✨ Visão Geral do Conteúdo

O PWA foi projetado seguindo as melhores práticas de **Ciclo de Vida de Automação**, **Processos de Qualidade** e **Desenvolvimento Ágil**, com deploy automatizado via GitHub Actions.

- **Acesso Offline:** Instale a aplicação no seu dispositivo para acesso sem internet.
- **Experiência Mobile-First:** UI adaptável e performática em qualquer tela.

## 🛠️ Stack Tecnológica Abordada

- **Linguagem:** Python (Otimização de GIL e gevent)
- **Web:** Selenium vs Playwright (CDP Protocol)
- **Mobile:** Appium (UIAutomator2 / XCUITest / ADB Shell)
- **Carga:** Locust (Distributed Mode / FastHttpUser)
- **Orquestração:** GitLab CI/CD (Runner Concurrency / DAGs)
- **Qualidade:** PWA, CI/CD, Processos Ágeis

## Configuração do Deploy
O deploy para o GitHub Pages está configurado via GitHub Actions no workflow `.github/workflows/deploy.yml`.
Ao fazer um push para a branch `main`, o PWA será buildado e publicado na branch `gh-pages`.
